# user-management-service
This project is a scalable User Management REST API developed using Spring Boot. 
It follows clean architecture principles and enterprise-level design practices.

Key features include CRUD operations, pagination, sorting, global search using JPA Specifications, 
DTO-based request/response handling, centralized exception management, and consistent API response structure.

- User CRUD operations (Create, Read, Update, Delete)
- Pagination, sorting, and global search
- JPA Specification-based dynamic filtering
- DTO pattern for request/response separation
- Global exception handling with structured error responses
- Centralized constants and reusable utilities
- Clean and scalable layered architecture
